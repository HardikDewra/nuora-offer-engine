Testing Methodology
How to run the next round of tests effectively

One Variable at a Time

Change exactly ONE element per test. All 7 previous tests changed 5+ variables, making it impossible to identify what specifically drives the difference.

Min 2,000 Visitors

Tests under 1,500 visitors have insufficient statistical power. At current traffic, aim for 12-15 hours runtime per test.

Always Run Concurrent Control

Control conversion varies 5.94% to 8.99% due to traffic quality. Never compare to historical numbers - always run simultaneously.

Track Sub Rate as Primary

A variant could match control on conversion but destroy subscription rate. For Nuora's model, sub rate is as important as CVR.
